# github-dns-revise

Use [ipaddress.com](https://www.ipaddress.com/) to solve github's CDN domain name pollution problem.

## Run

### Requirements

* requests
* lxml

### Usage

```bash
python main.py --f [host file]
```

Eaxmple in Windows 10:

```bash
python main.py --f C:/Windows/System32/drivers/etc/hosts
```
